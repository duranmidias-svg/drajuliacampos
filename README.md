# Landing Page - Dra. Júlia Campos

Arquivos prontos para subir no GitHub e publicar na Vercel.

## Estrutura

- `index.html`: página final com CSS, JS e imagens embutidos no próprio arquivo.

As imagens foram convertidas para Base64 dentro do HTML. A pasta `assets/` não é necessária para publicar esta versão.

## Publicação na Vercel

1. Envie o `index.html` para a raiz de um repositório no GitHub.
2. Na Vercel, importe o repositório.
3. Em `Framework Preset`, selecione `Other`.
4. Não precisa configurar build command.
5. Publique.
